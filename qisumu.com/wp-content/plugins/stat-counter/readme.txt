With Stat Counter Widget you can show to your users how much traffic you have on your wordpress site/blog. 

== Description ==

The counter is simple, it can track:

* Online Users
* Pageviews
* Unique Hits
* Online Users

You may also modify the Counter Widget from english to any language fast and easy.

If you need help with the plugin or you want to report a bug please refer to http://seo.uk.net/stat-counter/

== Installation ==

1. Upload the zip to 'plugins' directory or from WP Admin -> Plugins -> and search for stat counter.
1. Activate the plugin
1. You will be automatically redirected to your widgets page so you can configure the stat counter widget in your sidebar.

Stat Counter Widget does not have a settings section on Admin page. However, you can set the fields descriptions on the widget
